5. Criando a primeira tabela

Escrito por Uianes Luiz Rockenbach Biondo.
Última atualização em 04 out. 2023

Uma tabela em um "Banco de Dados" é composta por colunas e linhas. Em termos de "Banco de Dados", chamamos as colunas de ATRIBUTOS, enquanto as linhas podem ser chamadas de REGISTROS ou TUPLAS, sendo o segundo nome o mais comum.

Na etapa anterior, durante a diagramação, utilizamos o modelo relacional para representar a estrutura das tabelas, como exemplificado na imagem abaixo:

Tabela Cliente

PK cliente_id INT(4) NOT NULL
cliente_nome VARCHAR(100) NOT NULL

Nesse exemplo, temos os atributos "cliente_id" e "cliente_nome". As linhas são exemplos de tuplas, representando registros específicos da tabela. Cada linha (tupla/registro) é um cliente diferente cadastrado.

cliente_id cliente_nome
1001 Uianes
1002 Maria
1003 José
1004 Antônio
1005 João

Para criar a estrutura da tabela do exemplo apresentado, utilizamos as seguintes instruções na linguagem SQL:

CREATE TABLE Cliente(
cliente_id INT(4) NOT NULL,
cliente_nome VARCHAR(50) NOT NULL,
PRIMARY KEY(cliente_id)
);
