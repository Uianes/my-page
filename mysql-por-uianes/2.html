Considerando que o foco deste ano é o desenvolvimento de tecnologias web, iremos utilizar um servidor de Banco de Dados que permita a integração com o servidor Apache utilizado na disciplina de Programação II.

Portanto, existem várias opções: Apache XAMPP, EasyPHP e MySQL Workbench. Atualmente, o software XAMPP suporta a versão 8.0.21 do MySQL (versão atual desde 13 de julho de 2020) e vamos utilizá-lo para facilitar o desenvolvimento integrado com PHP.
