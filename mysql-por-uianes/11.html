11. Apagar uma tabela criada

Escrito por Uianes Luiz Rockenbach Biondo.
Última atualização em 04 out. 2023

Caso você tenha criado uma tabela e deseja apagá-la, basta utilizar o seguinte comando:

DROP TABLE nomeTabela;

Cuidado! Esse comando apagará a tabela e também todos os dados contidos nela.

Vale observar que você pode utilizar a seguinte estrutura para visualizar todas as tabelas de uma determinada base de dados:

SHOW DATABASES;
USE nomeDaBaseDeDados;
SHOW TABLES;

Isso mostrará todas as tabelas criadas em uma base de dados.
